# Interview Platform

A comprehensive AI-powered technical interview platform with video, audio, real-time chat, and collaborative code editing.

## Features

- **Live Video Interview**: Real-time video and audio using your camera and microphone
- **AI Interviewer**: Smart AI agent asks questions, evaluates answers, and provides hints
- **Code Editor**: Write and execute code in Python, Java, JavaScript, C, and C++
- **Real-time Chat**: Interactive chat interface with AI interviewer
- **Adaptive Difficulty**: AI adjusts question difficulty based on performance
- **Instant Feedback**: Detailed evaluation and feedback summary after each session

## Pages

- **index.html**: Landing page with features and how it works
- **dashboard.html**: Session management dashboard
- **interview.html**: Live interview room with video, chat, and code editor

## Technology

- React 18
- TailwindCSS
- Lucide Icons
